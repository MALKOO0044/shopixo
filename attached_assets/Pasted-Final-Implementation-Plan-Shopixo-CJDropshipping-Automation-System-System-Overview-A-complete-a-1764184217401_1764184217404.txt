Phase 5: Import to Shopixo
Time: Day 4

Import Process:

Admin clicks "Import Approved Products"
For each product, system:
Fetches full variant details from CJ (all colors, all sizes)
Applies KSA pricing to each variant
Downloads and stores product images
Generates unique Shopixo SKU: SPX-{CAT}-{SEQUENCE} (e.g., SPX-WBL-00142)
Checks for duplicates by CJ SKU
Creates product with all variants in Shopixo database
Logs success/failure to import_logs
Variant Data Captured:

Color name + color code
Size
Individual stock level
Variant-specific price if different
Variant SKU from CJ
Processing time (2-3 days)
Delivery estimate to KSA (7-15 days)
